# Asian kitchen

HTML, CSS ve Javascript ile asya yemeklerini listeleyen bir site yaptık.

Bu projede amaç bir obje içerisindeki verileri array ile dökümana yazdırmak ve çeşitli dizi metotlarını görmek.
